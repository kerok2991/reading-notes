## \<canvas\> and Chart.js

Canvas (`<canvas id='[id' width='[width]' height='[height]'>`) is an HTML5 element that graphics can be drawn onto. Chart.js is a library that utilizes the canvas element to make charts. In either case, it is useful to use javascript to get the element using `new context = document.getElementByID.getContext('2d')`, and from that context javascript can be used to draw: `context.moveTo(0, 0)` followed by `context.lineTo(100,10)` will draw a line that starts at 0, 0 (upper left corner) and goes to 10, 100 (100 pixels to the right, 10 pixels down). There's other shapes available in canvas, like `arc`, `filltext`, `stroketext`, `fillRect`, etc. There's transformations, colors/styles/shadows, line styles, paths, text, and rectangles. Here is a good cheat sheet: [https://simon.html5.org/dump/html5-canvas-cheat-sheet.html](https://simon.html5.org/dump/html5-canvas-cheat-sheet.html).

Chartjs worksin a similar way. For a line chart, can do: `new Chart(context, {type: [chartType], data [dataObject], options: [optionsObject]})` where `[dataObject]` is `{labels:[labelsArray], datasets:[{dataSet1},...,{dataSetn}]}`, where `{dataset}` contains the data array (as well as the series name "label", backgroundColor, borderColor) of the same length as the labels array. Other chart types can be made, such as bar charts, radar charts, doughnut/pie charts, polar area charts, bubble charts, scatter charts, area charts. The axes can either be radial or cartesian. Animating the charts is possible also (like when a new line series shows, it draws the line plot from left to right). It can also be responsive to mouse overs.
